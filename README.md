# Best VSCode settings file

### How to use
1. Open VSCode
2. Go to Command Palette (Cmd/CTRL + Shift + P)
3. Preferences: Open User Settings (JSON)
4. Paste **settings.json** content
5. Remove the settings that don't apply to your OS (i left a comment)
6. Install JetBrains Mono: https://www.jetbrains.com/lp/mono/ 
7. Install all required extension mentioned in **settings/json**
8. Restart VSCode
